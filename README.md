# Klaus-Dieter Löffler Fahrerservice

Statische Website für Krankenfahrten, Seniorenfahrten, Schülerfahrten, Kita-Fahrten und Chauffeurservice.

## Vercel

Das Projekt ist für Vercel ohne Build-Schritt vorbereitet.

- Framework Preset: `Other`
- Build Command: leer lassen
- Output Directory: leer lassen
- Install Command: leer lassen

Nach dem Verbinden mit dem GitHub-Repository deployt Vercel automatisch vom Branch `main`.

## Vor dem Livegang ersetzen

- Telefonnummer in `index.html`
- E-Mail-Adresse in `index.html`
- Domain in `index.html`, `robots.txt` und `sitemap.xml`
- Einsatzregion in den strukturierten Daten in `index.html`
